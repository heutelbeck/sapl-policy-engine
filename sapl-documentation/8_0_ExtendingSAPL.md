## Extending SAPL

SAPL can be extended with custom function libraries, custom attribute finders (PIPs), and custom PEP implementations for new frameworks and languages.

- **[PEP Implementation Specification](../8_1_PEPImplementationSpecification/):** The specification for building framework-integrated Policy Enforcement Points.
- **[Custom Function Libraries](../8_2_CustomFunctionLibraries/):** How to write custom function libraries that extend the SAPL policy language with new pure functions.
- **[Custom Attribute Finders](../8_3_CustomAttributeFinders/):** How to write custom Policy Information Points (PIPs) that provide attribute finders for accessing external data sources.
