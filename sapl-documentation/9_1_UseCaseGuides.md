## Use Case Guides

This section will provide end-to-end guides for common authorization scenarios.

### Planned Guides

- **REST API authorization:** Securing REST endpoints with SAPL policies
- **Microservice authorization:** Distributed authorization across microservice architectures
- **Time-based access control:** Policies that react to time boundaries, schedules, and shifts
- **Data filtering and transformation:** Using SAPL transformations to filter and redact response data
- **Multi-tenant access control:** Isolating tenants with SAPL policies and multi-subscription patterns

> **Planned content.** Each guide will include complete policy examples, integration code, and test strategies.
