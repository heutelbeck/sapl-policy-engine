## NestJS Integration

The SAPL NestJS SDK provides guards and decorators for integrating SAPL authorization into NestJS applications. It connects to a SAPL PDP server via the HTTP API and supports both streaming and one-shot authorization.

The SDK is available on npm as `sapl-nestjs`.
